HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains metadata that helps browsers render the page correctly.
  • HTML attributes define additonal information about an element.
  • Elements that are nested within other elements are called child elements.
  • The element that hold all the children elements is called the parent element.
  • Semantic HTML elements are elements described by its content and role in the webpage.
  • Properly ordering Headers is important for making the page accessible to users who access the screen using screen readers and other Assisitve technology.
  • Parent/child language is used by developers to explain relationship between elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which rules or declarations will apply.
  • Declarations contain two important components property and value.
  • Seperation of concerns is a term used to refer to seperating code into files based on use.
  • There are three ways to style a webpage , Inline , Internal and External being the preffered use.
